Optional adjustEstimate

adjustEstimate: string
  • Defined in version2/parameters/updateWorklog.ts:16

Defines how to update the issue's time estimate, the options are:

new Sets the estimate to a specific value, defined in newEstimate. leave Leaves the estimate unchanged. auto Updates the estimate by the difference between the original and updated value of timeSpent or timeSpentSeconds.
