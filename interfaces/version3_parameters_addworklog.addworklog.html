Optional adjustEstimate

adjustEstimate: string
  • Defined in version3/parameters/addWorklog.ts:15

Defines how to update the issue's time estimate, the options are:

new Sets the estimate to a specific value, defined in newEstimate. leave Leaves the estimate unchanged. manual Reduces the estimate by amount specified in reduceBy. auto Reduces the estimate by the value of timeSpent in the worklog.
